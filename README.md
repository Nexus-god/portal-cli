# Portal CLI

Быстрая установка CLI инструмента для Windows.

## Установка

```powershell
irm https://raw.githubusercontent.com/ТВОЙ_ЮЗЕР/portal-cli/main/install.ps1 | iex
```

Или через ваш сервер (когда nginx восстановится):
```powershell
irm http://195.226.92.230/install.ps1 | iex
```

## Использование

```bash
portalcli version
portalcli help
```

## Ручная установка

1. Скачать [portalcli.exe](https://github.com/ТВОЙ_ЮЗЕР/portal-cli/releases/latest/download/portalcli.exe)
2. Положить в `%LOCALAPPDATA%\portalcli\`
3. Добавить в PATH
