# Portal CLI

Быстрая установка CLI инструмента для Windows.

## Установка

```powershell
irm https://portalmodel.xyz/install.ps1 | iex
```

## Использование

```bash
portalcli version
portalcli help
```

## Ручная установка

1. Скачать [portalcli.exe](https://portalmodel.xyz/portalcli.exe)
2. Положить в `%LOCALAPPDATA%\portalcli\`
3. Добавить в PATH
